Starting off with how I made this sentence, I used the paragraph tag, which is < p > ,but without the spaces between.
The closing tag would be < /p > , also without the spaces.


To get the Header, you would use the tag < h > , and close it with < /h >.


To get the Navigation Bar, I used the tag, < ul > , which stands for "Unordered List", and would put items in the list by using the tag, < li > , which stands for "List Item"


In order to get the background image, I used the tag < img src= "URL_of_Picture" >.


To customize certain text, like the header, I used the < style > tag. For this tag to work, you have to set an ID for what you are trying to change.
You do that by adding < h id= "Hey"> This Is A Header < /h >.

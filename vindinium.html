Vindinium AI

Vindinium is a game, where you control a AI Bot, through code. In the game, there are mines and taverns, bots can own any mine, as long as it's free, and taverns are used to gain health. The player with the most gold at the end of the game wins.


The code used in Vindinium is JavaScript and some HTML. How you control your bot depends on the code you wrote.

Starting off with basic commands: To make your bot collect gold, you first have to create a variable and a task, and you have to set the task equal to a name we give it. We then create an if/else statement, for example, I have mine set to if my bot's health is below 45, it will go to a tavern.

The other if/else statement is that I made it so that my bot will go to a free mine, and claim it.


The code used to make the bot move different places is another if/else statement, which makes the bot go north, south, east, or west. based on myDir.
